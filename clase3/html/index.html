Curso Docker & Kubernetes — Clase 3

Nginx + MySQL + Adminer

Servidor web estático servido por nginx. Base de datos MySQL con volumen persistente.

Instrucciones rápidas

  1. Levantar: docker compose up -d
  2. Ver servicios: docker compose ps
  3. Acceder a la web: http://localhost:8080
  4. Adminer (GUI): http://localhost:8081

Información del proyecto

La BD de ejemplo contiene una tabla productos con datos insertados por initdb/init.sql.
